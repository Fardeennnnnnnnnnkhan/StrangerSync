# 🧬 StrangerSync

**Spontaneous Connections Redefined.**

StrangerSync is a premium, high-fidelity video chat protocol designed for instantaneous human connection. Built with a focus on privacy, security, and world-class aesthetics, it allows you to sync with random people across the globe through an encrypted, zero-log pipeline.

![StrangerSync Hero](https://images.unsplash.com/photo-1550751827-4bd374c3f58b?auto=format&fit=crop&q=80&w=1200)

## ✨ Core Pillars

### 🛡️ Privacy First (Zero Footprint)
We believe your identity shouldn't outlive your conversation. StrangerSync operates with a strictly **Zero Data Footprint** policy. No databases, no chat logs, and no historical tracking. Your session exists only as long as you are connected.

### ⚡ Nano-Latency Architecture
Engineered using custom **WebRTC P2P** tunnels and optimized **Socket.io** signaling, StrangerSync delivers high-definition video feedback with minimal latency, regardless of geographical distance.

### 🎨 Industry-Standard Aesthetics
Moving beyond generic web designs, StrangerSync features:
- **Dynamic Mesh Gradients**: Sophisticated background textures that react to your presence.
- **Glassmorphism UI**: High-fidelity surfaces with backdrop blurs and subtle noise overlays.
- **Orbital Connection Gallery**: A visually stunning homepage that represents global connectivity.
- **Micro-interactions**: Premium shimmer effects and elastic animations for a "living" interface.

## 🚀 Tech Stack

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/), [Framer Motion](https://www.framer.com/motion/), [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Backend**: [Node.js](https://nodejs.org/), [Express](https://expressjs.com/), [Socket.io](https://socket.io/)
- **Communication**: WebRTC (Peer-to-Peer)
- **Deployment**: [Render](https://render.com/)

## 🛠️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/fk748/StrangerSync.git
cd StrangerSync
```

### 2. Configure Backend
```bash
cd server
npm install
# Create a .env file and add your configuration
npm start
```

### 3. Configure Frontend
```bash
cd ../client
npm install
npm run dev
```
